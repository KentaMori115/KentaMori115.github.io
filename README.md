# Portfolio Website

A simple, clean portfolio website for Kenta Mori - Blockchain Engineer.

## Structure

- `index.html` - Main HTML file
- `style.css` - Styling and layout
- `script.js` - Interactive features and animations
- `README.md` - This file

## Features

- Clean, minimalist design
- Responsive layout
- Smooth scroll animations
- Professional color scheme with green accent
- Sections: Introduction, Profile, Career, Portfolio, Tech Stack, Education

## Deployment

To deploy on GitHub Pages:

1. Create a repository named `kenta-mori3322.github.io` on GitHub
2. Push these files to the repository
3. Enable GitHub Pages in repository settings
4. Your site will be live at `https://kenta-mori3322.github.io`

## Local Development

Simply open `index.html` in a web browser or use a local server:

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server
```

Then visit `http://localhost:8000` in your browser.
